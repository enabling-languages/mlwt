mlwt: LCG classes

CSS classes for use with LCG fonts developed by SIL International.

Language specific features

Vietnamese-style diacritics

Use language tagging

The Graphite and OpenType rules provide language specific features for Vietnamese. It is not necessarily to directly specify the graphite or opentype features for Vietnamese. The features will be activated automatically via language tagging.

To enable Vietnamese rendering add lang="vi" to the appropriate elements.

Some Latin script minority languages in Vietnam may also require Vietnamese typographic preferences. In this case either directly access the required graphite or opentype feature or write a CSS rule using a language pseudo-selector.

Serbian-style italic alternates

Use language tagging

The Graphite and Opentype rules provide language specific features for Serbian. It is not necessarily to directly specify the graphite feature for Serbian. The features will be activated automatically via language tagging. There is no direct opentype cvNN or ssNN feature for accessing Serbian italic variants.

To enable Serbian rendering add lang="sr" to the appropriate elements.

N.B. There is currently no support for Macedonian italic variants. When using Graphite rendering it is possible to access the serb feature.
